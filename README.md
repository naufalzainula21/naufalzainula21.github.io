# Naufal Zainul Arifin — Personal Portfolio Website

Live site: [naufalzainula21.github.io](https://naufalzainula21.github.io)

Personal portfolio website for Naufal Zainul Arifin, GIS Analyst and Data Engineer based in Bandung, Indonesia.

---

## Features

- **Dark mode** — toggle between light and dark theme, persisted via `localStorage`
- **Typewriter effect** — hero section cycles through roles (GIS Analyst, Data Engineer, etc.)
- **Animated stat counters** — years of experience, certificates, projects, and tools count up on load
- **Cursor spotlight** — subtle radial glow that follows the mouse
- **Active nav highlight** — navigation underline tracks the current section on scroll
- **Tech tag tooltips** — hover any tool in the stack to see a short description
- **Back-to-top button** — appears after scrolling down, smooth scroll back to top
- **Portfolio carousel** — horizontal sliding carousel with arrow buttons, dot indicators, swipe (touch), and keyboard (← →) support
- **Portfolio filter tabs** — filter projects by tag: SQL, GIS, Python, Database, Urban Planning, GitHub Repos
- **Fully responsive** — mobile-friendly layout, carousel shows 2 cards/page on mobile and 4 on desktop
